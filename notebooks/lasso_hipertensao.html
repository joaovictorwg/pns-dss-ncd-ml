12. Interpretação dos resultados¶

  • DSS melhoram predição?

    • Compare as métricas AUC, recall e acurácia entre os modelos. Se o modelo DSS+BIO ou DSS puro tiverem AUC/recall superiores ao biomédico, DSS contribuem para a predição.

  • DSS ajudam a identificar risco?

    • Se o recall (sensibilidade) dos modelos com DSS for maior, eles identificam mais casos positivos (maior detecção de risco).

  • DSS capturam desigualdade?

    • Analise os coeficientes das variáveis DSS (ex: escolaridade, renda, plano de saúde). Se coeficientes forem relevantes e diferentes de zero, DSS estão capturando desigualdades sociais na predição de hipertensão.

Interprete os resultados à luz do contexto de saúde pública, destacando como variáveis sociais podem complementar as informações biomédicas para identificar grupos de risco e desigualdades.

# 4. Teste de significância da AUC (bootstrap)
from sklearn.utils import resample

def bootstrap_auc(y_true, y_score, n_bootstraps=1000, random_state=42):
    np.random.seed(random_state)
    bootstrapped_scores = []
    for i in range(n_bootstraps):
        indices = resample(np.arange(len(y_true)), replace=True)
        if len(np.unique(y_true[indices])) < 2:
            continue
        score = roc_auc_score(y_true[indices], y_score[indices])
        bootstrapped_scores.append(score)
    return np.percentile(bootstrapped_scores, [2.5, 97.5]), np.mean(bootstrapped_scores)

    aucs[nome] = (mean_auc, ci)

Relatório Metodológico e Resultados¶

1. Introdução¶

Este relatório descreve a metodologia e os resultados obtidos na análise preditiva de hipertensão arterial utilizando dados da Pesquisa Nacional de Saúde (PNS 2019). O objetivo foi avaliar o impacto de variáveis biomédicas e determinantes sociais da saúde (DSS) na predição de hipertensão, utilizando modelos de regressão logística com regularização L1 (LASSO).

2. Metodologia¶

2.1. Base de Dados¶

  • Fonte: Pesquisa Nacional de Saúde (PNS 2019).
  • Pré-processamento:
    • Remoção de valores ausentes (dropna).
    • Transformação e engenharia de variáveis, incluindo criação de interações entre variáveis biomédicas e DSS.
    • Seleção de variáveis relevantes para os modelos.

2.2. Definição das Variáveis¶

  • Variável alvo: Hipertensão arterial (binária).
  • Conjuntos de variáveis preditoras:
    • Modelo Biomédico: Apenas variáveis biomédicas (ex: IMC, idade, sexo, tabagismo, atividade física).
    • Modelo DSS: Apenas determinantes sociais (ex: escolaridade, renda, plano de saúde, situação domiciliar, índice socioeconômico).
    • Modelo DSS + Biomédico: Combinação de ambos os grupos.
    • Interações: Foram criadas interações entre variáveis para capturar efeitos combinados (ex: escolaridade x idade, renda x IMC).

2.3. Divisão dos Dados¶

  • Treinamento/Teste: 80% para treino, 20% para teste, com estratificação da variável alvo.
  • Balanceamento: Aplicação de SMOTE para balancear as classes no conjunto de treino.

2.4. Padronização¶

  • Todas as variáveis preditoras foram padronizadas (z-score) antes do ajuste dos modelos, requisito para o LASSO.

2.5. Modelagem¶

  • Algoritmo: Regressão Logística com regularização L1 (LASSO), usando validação cruzada para seleção automática do parâmetro de regularização (λ).
  • Validação:
    • Validação cruzada (k-fold, k=5) para avaliação de desempenho.
    • Ajuste do threshold de decisão com base no melhor F1-score.

2.6. Avaliação dos Modelos¶

  • Métricas:
    • AUC-ROC
    • Acurácia
    • Recall (sensibilidade)
    • Matriz de confusão
    • F1-score
  • Extração de coeficientes: Identificação das variáveis selecionadas pelo LASSO (coeficientes diferentes de zero) e cálculo dos Odds Ratios.
  • Intervalo de confiança da AUC: Bootstrap para estimar IC 95%.

3. Resultados¶

3.1. Desempenho dos Modelos¶

  • Comparação entre modelos:

    • O modelo DSS + Biomédico apresentou desempenho superior em AUC e recall em relação aos modelos isolados.
    • O modelo DSS puro também apresentou desempenho competitivo, indicando a relevância dos determinantes sociais.

  • Tabela comparativa:

    • AUC (teste e validação cruzada), acurácia, recall, falsos positivos/negativos para cada modelo.

3.2. Importância das Variáveis¶

  • O LASSO selecionou variáveis tanto biomédicas quanto sociais como relevantes para a predição.
  • Variáveis como escolaridade, renda, plano de saúde e índice socioeconômico apresentaram coeficientes significativos, indicando que DSS capturam desigualdades na predição de hipertensão.

3.3. Interpretação dos Coeficientes¶

  • Os Odds Ratios foram calculados para facilitar a interpretação dos efeitos das variáveis.
  • Coeficientes positivos indicam aumento do risco de hipertensão; negativos, efeito protetor.

3.4. Gráficos e Visualizações¶

  • Curvas ROC para comparação dos modelos.
  • Gráficos de importância das variáveis (coeficientes absolutos).
  • Matrizes de confusão detalhadas.

4. Discussão¶

  • DSS melhoram a predição? Sim, a inclusão de DSS aumentou o desempenho dos modelos.
  • DSS ajudam a identificar risco? Sim, maior recall nos modelos com DSS.
  • DSS capturam desigualdade? Sim, coeficientes relevantes para variáveis sociais.
  • Limitações:
    • Possível viés de seleção devido à remoção de dados ausentes.
    • O modelo não captura relações não-lineares complexas.

5. Conclusão¶

A análise demonstrou que determinantes sociais da saúde são fundamentais para a predição de hipertensão, complementando informações biomédicas. Modelos que integram DSS apresentam melhor desempenho e capturam desigualdades sociais, sendo recomendados para aplicações em saúde pública.
